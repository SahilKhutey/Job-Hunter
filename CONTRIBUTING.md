# Contributing to AI Job Hunter OS

First off, thank you for considering contributing to Job Hunter OS! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## 🚀 How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issue Tracker.
- Describe the bug and include steps to reproduce.
- Mention your OS and browser version.

### Suggesting Enhancements
- Check the [ROADMAP.md](ROADMAP.md) first.
- Open a "Feature Request" issue.
- Explain why this enhancement would be useful.

### Pull Requests
1. Fork the repo.
2. Create a new branch (`git checkout -b feature/amazing-feature`).
3. Commit your changes (`git commit -m 'Add some amazing feature'`).
4. Push to the branch (`git push origin feature/amazing-feature`).
5. Open a Pull Request.

## 🛠️ Development Setup

Refer to the [README.md](README.md) for basic setup instructions. 

### Coding Standards
- **Python**: Follow PEP 8. Use docstrings for all services and API routes.
- **Frontend**: Use functional components with TypeScript. Prefer Tailwind for styling.
- **Commits**: Use [Conventional Commits](https://www.conventionalcommits.org/) (e.g., `feat:`, `fix:`, `docs:`).
