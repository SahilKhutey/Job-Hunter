# 🗺️ Product Roadmap: AI Job Hunter OS

The vision for **Job Hunter OS** is to move beyond "search and apply" into **Autonomous Career Execution**.

## 📍 Phase 1: The Trust Foundation (Current)
- [x] **Secure Identity**: JWT rotation, Google/LinkedIn OAuth.
- [x] **Profile Hydration**: AI-driven resume parsing and identity extraction.
- [x] **Tailoring Engine**: Per-job resume optimization and PDF generation.
- [x] **ATS Simulation**: Semantic matching and structural audit feedback.

## 🚀 Phase 2: Autonomous Execution (Next)
- [/] **Playwright Apply Agent**: Fully automated application form filling (Foundations implemented in `ExecutionAgent`).
- [ ] **Browser Fingerprinting & Persistence**: Dedicated browser contexts to maintain login state without triggering anti-bot measures.
- [ ] **Human-in-the-Loop (HITL) Dashboard**: Real-time observability of agent actions with a "Manual Override" toggle.
- [ ] **Multi-Agent Pipeline**: Parallel job scraping, matching, and tailoring.

## 🧠 Phase 3: Cognitive Intelligence (Q3 2026)
- [ ] **Interview Simulator**: AI-powered mock interviews based on the specific job description and user resume.
- [ ] **Negotiation Copilot**: Real-time salary research and response suggestion engine for recruiter emails.
- [ ] **Application Tracking System (ATS) for Users**: Visual Kanban board with automated status updates from email scraping.
- [ ] **Skill Gap Analysis**: Proactive suggestions for online courses/projects to increase match % for target roles.

## 🌐 Phase 4: Platform Expansion (Q4 2026)
- [ ] **Browser Extension**: 1-click tailoring while browsing any job site.
- [ ] **Mobile Mission Control**: iOS/Android app for real-time application alerts and one-tap approvals.
- [ ] **Enterprise/Team Version**: Collaborative job hunting for placement agencies.

---

*Note: This roadmap is subject to change based on community feedback and platform API constraints.*
